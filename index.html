Si₃N₄ Waveguide Design for Integrated Photonics

Indian Institute of Technology, Delhi

Lumerical MODE Lumerical FDTD
S Situation

Silicon Nitride (Si₃N₄) waveguides are critical building blocks for next-generation photonic integrated circuits due to their low optical loss and CMOS compatibility. The research group required optimized waveguide geometries for both visible (700nm) and telecom (1550nm) wavelengths to enable multi-application platform development.

T Task

Design and characterize Si₃N₄ waveguide structures that achieve optimal mode confinement while minimizing propagation losses. Subsequently, integrate these waveguides with high-Q ring resonators for sensing and filtering applications.

A Action

Utilized Lumerical MODE Solutions to perform eigenmode analysis, sweeping waveguide dimensions to map confinement factor vs. geometry. Characterized bending losses and coupling coefficients using FDTD simulations. Designed ring resonator structures with optimized gap spacing for critical coupling conditions.

R Result

Successfully delivered characterized waveguide designs with documented mode profiles, loss metrics, and ring resonator integration guidelines. The designs formed the foundation for the group's subsequent fabrication run.

High-Efficiency Hybrid Silicon-HfO₂-ITO Electro-Optic Modulator

Award-Winning Publication

Lumerical CHARGE INTERCONNECT
S Situation

Conventional silicon electro-optic modulators are limited by the weak plasma dispersion effect, requiring long device lengths or high voltages for adequate modulation depth. The photonics community actively seeks novel material platforms to achieve compact, low-power modulators for data center interconnects.

T Task

Design a hybrid modulator architecture leveraging Transparent Conducting Oxides (ITO) with epsilon-near-zero (ENZ) properties to achieve superior modulation efficiency compared to pure silicon approaches, while maintaining practical voltage requirements.

A Action

Developed a Si/HfO₂/ITO hybrid slot waveguide modulator architecture. Used Lumerical CHARGE to model carrier accumulation in the ITO layer and resulting permittivity modulation. Coupled electro-optical simulation to INTERCONNECT for system-level performance analysis including bandwidth estimation.

R Result

Achieved 1.07 nm/V modulation efficiency (among the highest reported for ITO-based modulators) and 11.12 dB extinction ratio at 4V. The work received the Best Research Paper Award at SAP 2025, IIT Indore.

Sunlight to Fiber Coupling System

API-Driven Optical Optimization

Zemax OpticStudio ZOS-API (Python)
S Situation

Coupling broadband sunlight into optical fiber systems is challenging due to the extended angular extent of the solar disk, chromatic aberration across the visible spectrum, and stringent alignment tolerances. Efficient solar-to-fiber coupling enables applications in daylighting, solar concentration, and spectroscopic systems.

T Task

Design an optimized optical coupling stage that maximizes power transfer from a sunlight collection system into a multimode optical fiber, surpassing baseline efficiency through systematic optical design optimization.

A Action

Leveraged the Zemax ZOS-API with Python to automate the optimization workflow. Explored PMMA ball lens configurations through programmatic parameter sweeps. Designed a hybrid optical stage incorporating an achromatic doublet to minimize chromatic focal shift. Implemented automated merit function evaluation for coupling efficiency.

R Result

Achieved ~12% improvement in coupling efficiency, reaching 75.98% overall efficiency. The Python automation reduced design iteration time from hours of manual GUI work to minutes of scripted optimization.

Automated Photonic Component Library

Python Module • October 2025

A custom Python library that automates two critical and time-consuming tasks in integrated photonics design:

1 Waveguide Dispersion Engineering Module

Calculates effective index, group index, and GVD as functions of waveguide geometry. Replaces manual parameter sweeps in Lumerical MODE with programmatic exploration.

2 Ring Resonator Coupling Calculator

Computes coupling coefficients and through/drop transmission for ring resonators. Eliminates the need for manual GUI estimation, providing instant analytical results.

from photonic_lib import WaveguideDispersion, RingCoupling

# Calculate dispersion for Si3N4 waveguide
wg = WaveguideDispersion(
    material="Si3N4",
    width=800e-9,  # 800 nm
    height=400e-9
)
n_eff, n_g, GVD = wg.calculate(wavelength=1550e-9)

# Estimate ring coupling
ring = RingCoupling(radius=20e-6, gap=200e-9)
kappa, through, drop = ring.analyze()
